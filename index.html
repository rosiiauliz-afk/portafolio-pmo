🔗 Configurar fuente de datos automática
¿Cómo funciona? Pega la URL pública de tu archivo Excel (o PPTX) en los campos de abajo. Al guardar, el portafolio cargará los datos automáticamente cada vez que se abra — sin que los demás usuarios tengan que subir nada.
💡 El portafolio convierte automáticamente links de Google Drive y SharePoint al formato correcto de descarga.
📖 ¿Cómo obtener la URL pública? (clic para ver)
Google Sheets / Drive: 1. Abre tu archivo en Google Sheets.
2. Menú Archivo → Compartir → Publicar en la web.
3. Elige la hoja, selecciona Valores separados por comas (.csv) o Microsoft Excel (.xlsx).
4. Haz clic en Publicar y copia el enlace.
También funciona con el link de exportación directa:
https://docs.google.com/spreadsheets/d/ID/export?format=xlsx
SharePoint / OneDrive: 1. Abre el archivo en SharePoint.
2. Haz clic en Compartir → Copiar vínculo.
3. Asegúrate de que el permiso sea "Cualquier usuario con el vínculo puede ver".
4. Pega el link aquí — el portafolio lo convertirá automáticamente al formato de descarga directa.
⚡ Flujo para compartir con tu equipo:
1. Pega la URL → Guardar y publicar → se descarga un nuevo HTML con la URL incrustada.
2. Sube ese HTML a Netlify (arrastra al dashboard).
3. Todos los que abran el link verán los datos automáticamente, siempre actualizados.
